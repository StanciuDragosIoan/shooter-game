hits(a, b) – Code + Explanation

function hits(a, b) {
  return Math.abs(a.x - b.x) < (a.w + b.w) / 2 &&
         Math.abs(a.y - b.y) < (a.h + b.h) / 2;
}
  • a.x - b.x → distance between centers on X axis
  • Math.abs() → makes distance always positive
  • (a.w + b.w) / 2 → how far apart they can be before touching
  • Same logic for Y axis
  • && → BOTH X and Y must overlap → real collision

Idea: Treat objects as rectangles from their center. If they overlap on both axes → collision.
